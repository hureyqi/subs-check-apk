# 产品需求文档 (PRD): Subs Check APK UI Mockup

## 概述
创建一个高质量的Android应用"Subs Check APK"UI模拟截图页面。该应用是一个代理订阅检查工具，采用Material Design 3设计风格。

## 目标
生成一个逼真的Android应用界面，展示应用的核心功能和视觉设计，可用于演示、宣传或设计参考。

## 用户场景
用户打开应用后，可以看到：
- 应用标题栏
- 订阅URL输入区域
- 检测进度指示
- 统计数据概览
- 运行日志
- 保存结果按钮

## UI组件需求

### 1. 应用栏/工具栏
- 应用标题："Subs Check"
- 右侧设置齿轮图标

### 2. URL输入卡片
- 圆角卡片设计
- 多行文本输入框，标签："请输入订阅 URL"
- 占位符文本显示多个订阅URL示例
- 底部按钮：主按钮"开始检测"(带播放图标)，次要设置图标按钮

### 3. 进度区域
- 线性进度条(约35%填充)
- 阶段文本："阶段: 测活"
- 统计文本："总计: 500, 已处理: 175"
- 轮廓"取消"按钮

### 4. 统计卡片
- 3列网格布局：
  - "总计": 500
  - "存活": 175 (绿色)
  - "最终": 89 (蓝色)

### 5. 日志区域
- 卡片包裹可滚动文本区域
- 等宽字体日志输出
- 显示时间戳和状态信息

### 6. 保存按钮
- 位于底部
- 文本："保存到 Download 文件夹"
- 带保存图标

## 设计要求
- Material Design 3 风格
- 紫色/靛蓝色主色调
- 浅色主题，干净白色背景
- 圆角卡片配微妙阴影
- 专业、现代的Android应用外观
- 居中显示，模拟手机屏幕
